◆ usage

std::string Command::usage

The usage of the command.

So that for a command that can be invoked as such : /msg Player123 This is a message The usage should be this : <Player:PlayerName> [quiet] <...message> The optional parameters being indicated with [param] and the mandatory ones with <param>. A parameter that takes all the remaining one should be listed as <...param> or [...param] if it is optional. The type is indicated after the :.
